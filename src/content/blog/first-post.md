## 起点

在信息过载的时代，安静本身就是一种表达。

当我开始设计这个博客时，我不想做又一个"现代化"的网站——渐变背景、动画过渡、卡片网格。那些很好，但不是我要的。

## 编辑设计的魅力

编辑设计（Editorial Design）源自印刷时代。书籍目录页、杂志排版，靠的是：

- **字体层级** — 大小、粗细对比建立信息结构
- **留白** — 不填满每一寸空间，让内容呼吸
- **引导线** — 点状线条连接条目与页码，功能即装饰

这些原则搬到网页上，不需要 JavaScript，不需要框架，只需要好的排版。

## 技术实现

这个网站用 Astro 构建，默认输出纯 HTML/CSS。点状引导线用 CSS flexbox 实现：

```css
.toc-item {
  display: flex;
  align-items: baseline;
}

.toc-dots {
  flex: 1;
  border-bottom: 1px dotted #c4b9a4;
}
```

简单，有效，优雅。

## 取舍

这种风格不追求视觉刺激。它安静、克制，甚至有点"无聊"。但对于一个以内容为核心的个人博客来说，安静就是最好的容器。
